## Stos Technologiczny

| Warstwa | Technologia |
|-------|-----------|
| **Frontend** | React + TypeScript |
| **Backend** | Node.js + Express |
| **Baza danych** | PostgreSQL / Azure SQL |

## Mapowanie Usług Azure

| Komponent | Usługa Azure | Cel |
|-----------|---------------|---------|
| Frontend | App Service / Static Web Apps | Hosting aplikacji React |
| Backend API | App Service | Hosting Node.js Express API |
| Baza danych | Azure SQL Database / Azure Database for PostgreSQL | Przechowywanie danych rezerwacji |
| Autentykacja | Azure AD | Zarządzanie autoryzacją użytkowników |
| Sieć | Virtual Network | Komunikacja między komponentami |
| Monitoring | Application Insights | Monitoring i diagnostyka |
